# 🎓 Student Performance Calculator

A simple web-based Student Performance Calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Enter marks of multiple subjects
- Calculate Total Marks
- Calculate Percentage
- Display Result Instantly
- Simple and User-Friendly Interface
- Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript`
- 
## 🎯 How It Works

1. Enter marks in the input fields.
2. Click the Calculate button.
3. The application calculates:
   - Total Marks
   - Percentage
   - Result
4. Output is displayed instantly on the screen.
